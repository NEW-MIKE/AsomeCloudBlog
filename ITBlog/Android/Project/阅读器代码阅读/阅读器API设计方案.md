### 工作流程：

打开书籍，输入书籍的对象，通过反序列化来实现。然后看看本书是否还存在，存在的话，就打开。不存在的话就提示，然后不进入阅读界面。如果存在，就传递一个信息给阅读的界面，你给我一个地址，我就通过这个地址打开一个页面，在页面里面通过地址打开，然后是当前阅读的界面的阅读的历史。
